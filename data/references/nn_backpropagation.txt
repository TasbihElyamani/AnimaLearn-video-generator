Backpropagation: The Core Training Algorithm

Backpropagation (backprop) efficiently computes gradients of the loss function with respect to network weights using the chain rule. Forward pass: input flows through layers, producing predictions. Loss computation: compare predictions to targets. Backward pass: gradients flow backward from loss to each weight. For a simple network: dL/dw = dL/da · da/dz · dz/dw where L=loss, a=activation, z=pre-activation, w=weight. Computational graphs track operations for automatic differentiation. Frameworks (PyTorch, TensorFlow) implement autograd for automatic backpropagation. Gradient accumulation handles large batch sizes on limited memory. Gradient clipping prevents exploding gradients. Mixed-precision training uses float16 for speed with float32 for stability.
