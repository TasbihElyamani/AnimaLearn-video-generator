Neural Network Applications

Computer vision: image classification (ResNet), object detection (YOLO, Faster R-CNN), image segmentation (U-Net), face recognition. Natural language processing: machine translation (seq2seq, Transformers), sentiment analysis, text generation (GPT), question answering (BERT). Speech: recognition (DeepSpeech, Whisper), synthesis (WaveNet, Tacotron). Autonomous vehicles use CNNs for perception and RNNs for planning. Healthcare: medical image diagnosis (detecting tumors, retinal diseases), drug discovery (molecular property prediction), protein folding (AlphaFold). Gaming: AlphaGo (2016) beat the world champion at Go using deep reinforcement learning. Recommendation systems (Netflix, Spotify) use embeddings and collaborative filtering. Generative AI: DALL-E, Stable Diffusion, ChatGPT, Midjourney.
